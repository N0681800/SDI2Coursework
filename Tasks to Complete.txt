Analysis and Design
The project should contain the following design and analysis for the WHOLE software systems:

* The Class diagram

diagram.

* Deployment diagrams.
* Any design patterns

i.e., identify possible architecture styles and choose one with respect to the identified utility tree,
you need to explain the reason)
